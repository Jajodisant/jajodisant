# 🚀 Proyectos Destacados

## 🧠 LibroTech — Catálogo Inteligente con IA
Desarrollo de un catálogo inteligente de libros que integra modelos de Inteligencia Artificial para interactuar con los usuarios en tiempo real.
- **Características:** Integración de IA con **Google Gemini** (vía Spring AI y LangChain4J). Comunicación bidireccional y en tiempo real usando **WebSockets**[cite: 1]. Arquitectura con persistencia políglota[cite: 1].
- **Stack:** Java 17 • Spring Boot • Spring AI • PostgreSQL • MongoDB • WebSockets[cite: 1].
- 🔗 **[Ir al repositorio](https://github.com/Jajodisant/LibroTechRepo.git)** 

## 🏨 HotelNovaSystem — Sistema de Gestión Hotelera
Diseño y construcción de un sistema backend robusto y automatizado para la administración hotelera[cite: 1].
- **Características:** Conexión y modelado de datos optimizado con bases de datos relacionales[cite: 1]. Pruebas unitarias automatizadas para asegurar la estabilidad del negocio[cite: 1]. Empaquetado independiente mediante Maven Shade[cite: 1].
- **Stack:** Java 21 • Spring Boot • JPA/Hibernate • PostgreSQL • JUnit 5 • Maven[cite: 1].
- 🔗 **[Ir al repositorio](https://github.com/Jajodisant/hotalNovaSystem.git)**

## 🐍 API Backend con FastAPI
API REST robusta diseñada para la gestión segura de usuarios y productos comerciales[cite: 1].
- **Características:** CRUD completo con inyección de dependencias, hashing de contraseñas y arquitectura limpia. Autenticación y autorización basada en **JWT (JSON Web Tokens)**[cite: 1].
- **Stack:** Python • FastAPI • REST API • JWT • Pydantic[cite: 1].

## 📊 Análisis Exploratorio de Datos — House Pricing
Proyecto de análisis de datos enfocado en la limpieza y modelado estadístico para la predicción de variables[cite: 1].
- **Características:** Limpieza de datos (data cleaning), manejo de valores nulos, transformaciones lógicas de variables, análisis estadístico y visualización gráfica[cite: 1].
- **Stack:** Python • Pandas • NumPy • Matplotlib • Seaborn • Jupyter Notebook[cite: 1].
